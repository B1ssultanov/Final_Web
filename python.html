ПРО PYTHON

Python — высокоуровневый язык программирования общего назначения
с динамической строгой типизацией и автоматическим управлением памятью,
ориентированный на повышение производительности разработчика, читаемости кода и его качества,
а также на обеспечение переносимости написанных на нём программ.
Язык является полностью объектно-ориентированным в том плане, что всё является объектами.
Необычной особенностью языка является выделение блоков кода пробельными отступами.

ПЛЮСЫ

  • Простой синтаксис для визуального восприятия кода;
  • Легкий для изучения;
  • Изобилие фреймворков и библиотек;
  • ООП;
  • Кроссплатформенность;

МИНУСЫ

  • Один из самых медленных языков программирования;
  • Не подходит для задач с большим объемом памяти;

Типы данных

int

Целое, целочисленный тип данных — один из самых простых примитивных типов данных. Служит для представления целых чисел, ограниченного минимальным и максимальным значением, зависящими от выделенной под число памяти.

float

Число с плавающей запятой — экспоненциальная форма представления вещественных чисел, в которой число хранится в виде мантиссы и порядка

bool

Логи́ческий тип да́нных — примитивный тип данных в информатике, принимающий два возможных значения, иногда называемых истиной (true) и ложью (false)

string

Строковой тип данных - тип данных, значениями которого является произвольная последовательность (строка) символов алфавита.

if

Реализует выполнение
определённых команд при условии,
что некоторое логическое
выражение принимает значение «истина»

elif

Оператор elif управляет условным ветвлением.
Операторы выполняются только в том случае,
если condition вычисляется ненулевое значение. Если значение condition ненулевое, выполняется следующая инструкция

else

Оператор else выполняется
когда условия всех
других операторов
не могут быть выполнены

Операторы цикла

Операторы цикла while и for были созданы для того чтобы
определенное действие повторялось до тех пор, пока выполняется условие
