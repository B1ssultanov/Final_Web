ПРО PYTHON

Python — высокоуровневый язык программирования общего назначения
с динамической строгой типизацией и автоматическим управлением памятью,
ориентированный на повышение производительности разработчика, читаемости кода и его качества,
а также на обеспечение переносимости написанных на нём программ.
Язык является полностью объектно-ориентированным в том плане, что всё является объектами.
Необычной особенностью языка является выделение блоков кода пробельными отступами.

ПЛЮСЫ

  • Простой синтаксис для визуального восприятия кода;
  • Легкий для изучения;
  • Изобилие фреймворков и библиотек;
  • ООП;
  • Кроссплатформенность;

МИНУСЫ

  • Один из самых медленных языков программирования;
  • Не подходит для задач с большим объемом памяти;
